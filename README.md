# Odisha Tourism Website

A modern, responsive tourism website for exploring the rich cultural heritage and beautiful destinations of Odisha, India. Built with HTML and CSS.

## Overview

This project showcases the best of Odisha's tourism offerings, including famous temples, luxury hotels, cultural activities, and travel experiences. The website provides visitors with comprehensive information to plan their trip to Odisha.

## Features

- **Popular Destinations**: Explore iconic locations including Jagannath Temple (Puri), Lingaraj Temple (Bhubaneswar), and Dhauli Giri
- **Featured Hotels**: Information about premium accommodations including Swosti Premium, Mayfair, and Sterling Resort
- **Cultural Activities**: Discover Odisha's heritage through forest tours, cultural excursions, and traditional Odishi dance
- **Team Section**: Meet the dedicated team behind Odisha Tourism
- **Contact Form**: Easy way for visitors to get in touch for inquiries and bookings
- **Responsive Design**: Fully responsive layout that works on all devices

## Technologies Used

- **HTML5**: Semantic and accessible markup
- **CSS3**: Modern styling with responsive design
- **Images**: Optimized image assets for destinations, hotels, and activities

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the various sections using the navigation menu
